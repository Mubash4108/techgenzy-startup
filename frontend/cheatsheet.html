🚀 Web Development Cheatsheet

Copy & paste these code snippets when building projects

HTML - Basic Tags

Full Page Template:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Page</title> <link rel="stylesheet" href="style.css"> </head> <body> <!-- Your content here --> <script src="script.js"></script> </body> </html>

Common Tags:

<h1> Main Heading </h1> <h2> Sub Heading </h2> <p> Paragraph text </p> <a href="https://example.com"> Link </a> <button> Click Me </button> <img src="image.jpg" alt="Description"> <input type="text" placeholder="Enter text"> <ul> <li> Item </li> </ul>

CSS - Styling

Color Examples:

/* By name */ color: red; /* Hex codes (0-9, A-F) */ color: #FF0000; /* RGB */ color: rgb(255, 0, 0); /* Check: htmlcolorcodes.com */

Text Styling:

color: #333; /* Text color */ font-size: 18px; /* Size */ font-weight: bold; /* Bold */ font-family: Arial; /* Font type */ text-align: center; /* center/left/right */ text-decoration: underline; line-height: 1.6; /* Space between lines */

Box Model (Spacing):

padding: 20px; /* Space INSIDE */ margin: 20px; /* Space OUTSIDE */ border: 2px solid #ccc; /* Border line */ border-radius: 8px; /* Rounded corners */ width: 100%; /* Width */ height: 100px; /* Height */

Hover Effects:

button:hover { background-color: #007bff; cursor: pointer; transform: scale(1.05); }

Responsive (Mobile):

@media (max-width: 600px) { body { font-size: 14px; } .container { width: 90%; } }

JavaScript - Logic

Variables (Store Data):

let name = "John"; // Can change const age = 25; // Cannot change let counter = 0; let items = [ "apple", "milk" ]; /* Array */

Functions (Reusable Code):

function greet(name) { console.log("Hello " + name); } greet("John"); /* Arrow function (modern) */ const add = (a, b) => a + b;

Get HTML Elements:

/* By ID */ let element = document.getElementById("myId"); /* By class */ let elements = document.querySelectorAll(".myClass"); /* Change content */ element.innerHTML = "New text"; element.textContent = "New text"; /* Change style */ element.style.color = "red"; element.style.display = "none"; /* Hide */

Events (React to Clicks):

/* In HTML */ <button onclick="myFunction()"> Click </button> /* In JavaScript */ document.getElementById("btn").addEventListener("click", function() { alert("Clicked!"); });

Conditions (If/Else):

if (age >= 18) { console.log("Adult"); } else if (age >= 13) { console.log("Teen"); } else { console.log("Child"); }

Loops (Repeat Code):

/* Loop through array */ let fruits = ["apple", "banana"]; fruits.forEach(fruit => { console.log(fruit); }); /* Traditional loop */ for (let i = 0; i < 5; i++) { console.log(i); }

Array Methods:

let arr = [1, 2, 3]; arr.push(4); /* Add item */ arr.pop(); /* Remove last */ arr.length; /* Count items */ arr.includes(2); /* Check if exists */ arr.map(x => x * 2); /* Transform items */ arr.filter(x => x > 1); /* Keep matching */
