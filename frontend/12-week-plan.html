🗓️ Your 12-Week Web Development Roadmap

Goal: Build real skills for a junior developer job

Time Commitment: 30 minutes daily (Can work! Check schedule below)

📚 PHASE 1: HTML BASICS (Weeks 1-4)

Focus: Learn website structure

Days 1-2: HTML Introduction
✓ Watch: HTML in 1 hour (YouTube: Traversy Media)
✓ Read: MDN - What is HTML?
✓ Create: Your first HTML file with title, heading, paragraph
Days 3-4: HTML Tags
✓ Learn: p, h1-h6, a, img, button, ul, li
✓ Create: Page with list, links, and image
✓ Practice: Make 2 different pages
Days 5-7: Forms & Inputs
✓ Learn: input, textarea, form, label
✓ Create: Simple contact form
✓ Practice: Create a registration form
Days 8-14: Practice Week
✓ Create 3 pages: About me, Services, Contact
✓ Use all HTML tags you learned
✓ Make sure pages are well-organized
🏆 Phase 1 Challenge: Build a 3-page website about any topic (pets, cars, food). Use only HTML, no styling yet!
✅ CHECKPOINT 1: You should understand HTML structure well
Know 15+ HTML tags by heart

🎨 PHASE 2: CSS STYLING (Weeks 5-8)

Focus: Make pages beautiful

Days 15-16: CSS Introduction
✓ Watch: CSS in 1 hour (YouTube: Traversy Media)
✓ Learn: What is CSS, how to link to HTML
✓ Create: Simple colored page
Days 17-21: Styling Basics
✓ Learn: Colors, fonts, padding, margin
✓ Create: Style your Phase 1 pages
✓ Experiment: Try 5 different color schemes
Days 22-28: Layout & Flexbox
✓ Learn: Flexbox (most important!)
✓ Watch: Flexbox in 20 mins
✓ Create: Flex layout for header, content, sidebar
✓ Practice: Align items in different ways
Days 29-30: Hover & Effects
✓ Learn: :hover, transform, transition
✓ Create: Buttons that change on hover
✓ Add: Smooth animations
🏆 Phase 2 Challenge: Take your Phase 1 website and style it beautifully. Should look professional!
✅ CHECKPOINT 2: Your pages should look nice
Responsive design (works on mobile too)

⚡ PHASE 3: JAVASCRIPT INTERACTIVITY (Weeks 9-12)

Focus: Make things interactive

Days 31-32: JavaScript Basics
✓ Watch: JavaScript 1-hour intro
✓ Learn: Variables, console.log, data types
✓ Practice: 10 console.log exercises
Days 33-35: Functions & Events
✓ Learn: Functions (how to write & use)
✓ Learn: onclick, addEventListener
✓ Create: Button that shows message on click
✓ Practice: Make 5 different button interactions
Days 36-42: DOM Manipulation
✓ Learn: getElementById, querySelector
✓ Learn: innerHTML, textContent, style
✓ Create: Change page content dynamically
✓ Practice: Toggle visibility, change colors
Days 43-46: Loops, Arrays, Conditions
✓ Learn: if/else statements
✓ Learn: for loops, while loops
✓ Learn: Arrays (.push, .pop, .map, .filter)
✓ Practice: Loop exercises daily
Days 47-56: Build Projects
✓ Week 1: Counter App (++) ✓✓✓
✓ Week 1: Color Changer
✓ Week 2: Todo List (Add/Delete/Mark Done)
✓ Week 2: Simple Quiz Game
✓ Final: Calculator or Weather App
🏆 Phase 3 Challenge: Build a fully working Todo List or Calculator app. Should store data and handle errors!
✅ CHECKPOINT 3: Can build interactive, working apps
Ready to learn frameworks (React)

🚀 AFTER 12 WEEKS: Next Steps

What You Have Now:

  • ✅ Strong HTML/CSS/JavaScript skills
  • ✅ 5+ projects in your portfolio
  • ✅ Can build interactive websites
  • ✅ Can debug code independently

Build servers and databases
Time: 4-5 weeks learning + 2 weeks building projects
Option C: Keep Improving Frontend
Master TypeScript, testing, more projects
Build 3-5 impressive portfolio projects

3. Deploy to Netlify/Vercel (free)
4. Write project descriptions
5. Share your portfolio link
6. Create resume with projects

Start Job Hunt:

✓ Look at: LinkedIn, Indeed, AngelList
✓ Have: Portfolio, Resume, GitHub ready
✓ Practice: Common interview questions
