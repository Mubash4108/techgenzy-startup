Step 1: Install Git

Git is version control software (free).

Option A: Download Git (Recommended)

Go to: git-scm.com/download/mac

Download and install the .dmg file

Option B: Use Terminal (Advanced)

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install git

Step 2: Create GitHub Account

Go to: github.com

Click "Sign up" → Use your email → Create username

Username suggestion: yourname-dev or mubasheer-web

Step 3: Create New Repository

  1. Click the "+" icon → "New repository"
  2. Repository name: techgenzy-startup
  3. Description: TechGenzY - Web Development Learning Platform & Startup
  4. ✅ Make it Public (so people can see your work)
  5. ❌ Don't check "Add a README file"
  6. Click "Create repository"

Step 4: Upload Your Files

Method A: Upload via GitHub Website (Easiest)

  1. On your repository page, click "Add file" → "Upload files"
  2. Drag all your HTML files from the frontend folder
  3. Add commit message: "Initial commit: TechGenzY startup website"
  4. Click "Commit changes"

Method B: Using Git Commands (Advanced)

cd /Users/mubasheer/Documents/java/frontend
git init
git add .
git commit -m "Initial commit: TechGenzY startup website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/techgenzy-startup.git
git push -u origin main

Replace YOUR_USERNAME with your actual GitHub username

Step 5: Check Your Repository

Go back to github.com/YOUR_USERNAME/techgenzy-startup

You should see all your files listed!

🎉 Success! Your code is now on GitHub!

Step 6: Auto-Deploy to Netlify (Optional)

  1. Go to netlify.com
  2. Click "Add new site" → "Import from Git"
  3. Connect your GitHub account
  4. Select "techgenzy-startup" repository
  5. Deploy automatically!

Benefit: Every time you update code on GitHub, Netlify redeploys automatically!

Step 7: Share Your Work

❌ Common Issues & Solutions

Git Not Found:

Download from git-scm.com and install

Permission Denied:

Make sure you're logged into GitHub

Files Too Big:

GitHub has 100MB limit per file. Your HTML files are fine!

Forgot Commands:

Use the website upload method - it's easier for beginners
